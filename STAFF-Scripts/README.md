# Legacy scripts

This folder contains the original scripts from the fork history. They are kept for provenance and for anyone who needs to inspect the earlier dataset workflow.

The active STAFF public artifact is the sanitized `gpt-5.5` results bundle under `public-results/MBPP-DFY-153-fixed-gpt55-full`.

The historical scripts still translate Dafny into C++ and run the older dataset pipeline.
